# Web Research Protocol

Every technical/design/scientific phase must include current web research.

Required source preference:
1. Official documentation.
2. Maintained open-source repositories.
3. Peer-reviewed or standards documents.
4. Current high-quality implementation articles.

Each research note must include:
- query/topic;
- sources consulted;
- source quality;
- adopted decisions;
- rejected approaches;
- risks.

No agent may say “current best practice” without recorded research.
