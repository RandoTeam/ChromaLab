# Test Matrix Requirements

Dataset must include:

1. Clean screenshot.
2. Phone screenshot.
3. Already-cropped chart panel.
4. Graph with title/ion near top.
5. Graph with labels inside plotArea.
6. Dense peak series.
7. Weak/faint peaks.
8. Perspective-distorted photo.
9. Missing/partial tick labels.
10. Real multi-graph page.
11. One physical graph with many ROI candidates.
12. High text contamination case.

For each image define expected behavior for:

- AUTO_DIAGNOSTIC;
- GUIDED_PRODUCTION;
- MANUAL_ADVANCED.
