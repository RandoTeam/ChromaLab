## 1.1 — Базовая цель фазы

- [x] Реализовать путь: фото → выравнивание → выбор графика → калибровка осей → извлечение кривой → массив точек
- [x] На этом этапе не делать финальные экспертные выводы
- [x] На этом этапе не делать автоматическое определение вещества
- [x] Основная цель — проверить, можно ли детерминированно получить пригодный цифровой сигнал из фото
- [x] Все результаты по фото помечать как расчетные/ориентировочные до валидации
