# External Graph Fixture Candidates

Status: candidate catalog only.

This directory tracks external graph sources discovered during deep research waves. It is intentionally not a dumping ground for downloaded images.

Rules:

- Do not commit external images unless license, attribution, and intended use are documented.
- Prefer small curated subsets over full datasets.
- Keep chromatogram fixtures separate from generic chart fixtures.
- Generic chart fixtures may be useful for axis/OCR/layout stress tests, but they do not prove chromatogram scientific correctness.
- Any future imported image must include source URL, license, attribution, expected graph count, and why it belongs in ChromaLab validation.
